# Foundation

> **Anime Calendar v2 is an open-source Anime Release Intelligence Platform dedicated to accuracy, transparency, personalization, and long-term maintainability.**

## Why we exist

Anime fans should not need to search multiple sites to answer simple questions:

- What anime is releasing?
- When will it be available?
- Where can it be watched legally?
- Which version—original, sub, dub, movie, OVA, or special—is available?
- How certain is the information?

Anime Calendar v2 exists to answer those questions from one trusted, understandable source.

## What we believe

### Truth over guesses

Confirmed information and predictions are different products. The project must never present an estimate as a fact. Unknown is a valid and often preferable answer.

### Transparency earns trust

Important release and provider claims should carry evidence, confidence, and source context. Users should be able to understand why the system believes something.

### Respect the user's time

The project should reduce searching, duplicate notifications, and unnecessary decisions. Outputs should be useful rather than engagement-driven.

### User control is fundamental

Users decide which genres, providers, formats, titles, versions, calendars, and notifications matter to them.

### Maintainability is a feature

Readable code, tests, documentation, predictable architecture, and explicit decisions are part of the product—not administrative overhead.

### Open systems create durable value

Public metadata models, documented architecture, portable user data, and reusable output interfaces make the project stronger and easier to trust.

### Build for longevity

We are not building the fastest anime tracker. We are building the one we would be proud to recommend to anyone. Major decisions should still make sense years from now.

## What we are building

Anime Calendar v2 is:

- an anime release intelligence engine;
- a streaming-provider intelligence engine;
- a source-aware confidence and evidence system;
- a generator for calendars and future output plugins;
- a foundation for personalized feeds, dashboards, APIs, and notifications;
- a contributor-friendly open-source project.

## What we are not building

Anime Calendar v2 is not:

- a piracy or torrent directory;
- an unauthorized streaming proxy;
- a rumor mill;
- a clickbait news product;
- an engagement-maximization platform;
- a system that hides uncertainty to appear complete.

## The manifesto

We believe anime fans deserve a trusted source of release information.

We believe accuracy matters more than speed.

We believe transparency builds trust.

We believe software should empower users rather than compete for their attention.

We believe maintainability is a feature.

We believe every contribution should leave the project better than it was found.

We believe that when we do not know something, we should say so.

And above all:

> **Anime fans should spend their time watching anime—not trying to determine when, where, or which version is available.**
